API Яндекс карт

Написать функцию storeMapInit, которая определяет поведение карты 
и реакцию на действия пользователя:

 - при загрузке страницы: 
   -- карта открывается с текущего выбранного города,
   -- масштаб карты оптимален для этого города,
   -- в качестве параметра, определяеющего положение карты используется название города

 - на карту передеаются метки всех магазинов всех городов
 - при клике на метку карта приближается до здания/строения на котором стоит метка, появляется модальное окно с информацией о магазине
 - при клике на "крестик" модальное окно закрывается, масштаб карты изменяется до города
 - на карте активирован zoom с помощью колеса мыши

Общий пример страницы: https://d05.zoloto585.ru/store/
в архиве: 
 - файл "shops.js" содержит данные по магазинам
 - файл "store-item.html" содержит шаблон разметки модального окна с информацией о магазине
